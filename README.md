# Space_Invaders

This was my largest project yet. I followed a tutorial by @TokyoEdTech on YouTube on how to make a space invader game using the turtle module. After creating the game, I challenged myself to use the knowledge I had learned from the video implement my own ideas: to fire two bullets instead of one, add a "losing" sound and a "winning sound", lose if the enemy hits the lower most wall, and incease enemy speed everytime an enemy dies.

Press the left and right arrows, or the keys 'A' and 'D', to move left and right. Press the spacebar to shoot bullets. If the invaders hit the ship or the boundary behind you, you lose. If you defeat all of the invaders, you win!

I learned a lot from this project! I hope you enjoy.
